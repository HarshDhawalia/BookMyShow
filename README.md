# BookMyShow Backend

Spring Boot backend service for a BookMyShow-like movie booking system.

## Tech Stack
- Java 18
- Spring Boot 4
- Spring Data JPA
- Hibernate
- MySQL
- Maven

## How to Run
1. Start MySQL
2. Create database `BookMyShow`
3. Configure `application.properties`
4. Run:
   ```bash
   mvn spring-boot:run
